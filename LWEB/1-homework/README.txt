Descrizione:

L'esercizio da cui abbiamo preso spunto è il "CSS-3" presente nella slide 41 del seguente pdf "lweb-part02-XHTML.CSS.2.pdf". 

Abbiamo scelto di ampliarlo inserendo diverse pagine XHTML, con rispettivi file CSS, e cambiando il topic principale dell'esercizio scegliendo come soggetto un blog in cui vengono recensiti libri. 

Svolgendo l'esercizio i tag che sono stati utilizzati maggiormente sono:

- <div>, <ul>, <li> e <table> per organizzare i vari elementi presenti nelle pagine XHTML

- <a> per linkare le varie pagine XHTML tra loro e per i collegamenti alle pagine esterne

- <img> per arricchire il contenuto del sito tramite delle immagini 



Mentre le proprietà principali da noi utilizzate nei file CSS sono:

- font-family: per specificare il font di un certo elemento

- font-style: per ulteriori specifiche al font di un certo elemento

- font-size: per regolare la dimensione del testo

- text-align: per posizionare il testo secondo un certo ordine

- padding, margin, display: per posizionare gli elementi in maniera ordinata nelle varie pagine
	
	- nello specifico il file "consigliati.html" è stato strutturato tramite il "Grid Layout", presente nel selector ".grid".
